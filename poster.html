HOW TO START
3단계로 시작하기
1
안전교육이수증 업로드
로그인 화면의 'SIGN UP' 클릭 → 학번, 이름, 학년, 전공, 이메일 입력 → 4자리 비밀번호 설정 → 이수증 파일 제출
→
2
로그인
학번, 이름, 4자리 비밀번호 입력 → '로그인' 버튼 클릭
💡 '로그인 기억' 체크 시 자동 로그인
→
3
서비스 이용
대시보드에서 현황 확인 → 원하는 메뉴(예약/대여/출력) 선택 → 신청 완료!
FEATURES
주요 기능
🏠
실기실 예약
모형제작실, 3D프린팅실, 캐드실, 레이저커팅실 등 예약
🔧
물품 대여
전동공구, 수공구, 계측장비 등 실시간 재고 확인 후 대여
🖨️
출력 신청
Coated, Matt, Gloss, 흑백, 컬러 등 다양한 출력 요청
📊
대시보드
예약, 대여, 출력, 문의 현황을 한눈에 확인
📋
양식함
필요한 양식 파일을 플로팅 버튼으로 간편 다운로드
💬
문의하기
포털에서 바로 문의 작성, 답변 확인 가능
LOGIN GUIDE
역할별 로그인
👩‍🎓
학생 로그인
1
최초 1회 안전교육이수증 업로드 (SIGN UP)
2
학번 + 이름 + 비밀번호(4자리) 입력
3
로그인 버튼 클릭 → 학생 포털 진입
🔧
근로학생 / 관리자 로그인
1
상단 탭에서 '근로학생' 또는 '관리자' 선택
2
부여받은 아이디 + 비밀번호 입력
3
관리 화면 접속 → 예약/대여/출력 처리
⚠️ 주의사항
안전교육 미이수 시 실기실 예약·물품 대여 불가. 반드시 이수증을 먼저 업로드하세요. 반납기한 초과 시 연체 경고 부여, 블랙리스트 등록 시 포털 이용 제한.
💡 이런 것도 가능해요!
학생증 사진 변경 — 로그인 화면에서 바로 신청 │ 해동예약포털 — 사이드바에서 해동KL 장비 예약 │ 다크모드 — 🌙 버튼으로 전환
FAQ
자주 묻는 질문
🔐 비밀번호를 잊었어요
이수증 업로드 시 설정한 4자리 숫자입니다. 이수증을 다시 업로드하여 재설정하세요.
🚫 이수증 후 로그인 안 돼요
관리자 승인이 필요합니다. 교학팀(복지관 602호)에 문의하세요.
📦 물품 재고가 0이에요
다른 학생의 반납을 기다리거나 근로학생에게 직접 문의해 주세요.
🖨️ 출력물 처리 시간은?
근무시간 내 순차 처리됩니다. '출력 신청' 탭에서 진행 상태를 확인하세요.
