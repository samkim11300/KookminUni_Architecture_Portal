HOW TO START
3단계로 시작하기
1
안전교육이수증 업로드
로그인 화면의 'SIGN UP' 클릭 → 학번, 이름, 학년, 전공, 이메일 입력 → 4자리 비밀번호 설정 → 이수증 파일 제출
→
2
로그인
학번, 이름, 4자리 비밀번호 입력 → '로그인' 버튼 클릭
💡 '로그인 기억' 체크 시 자동 로그인
→
3
서비스 이용
대시보드에서 현황 확인 → 원하는 메뉴(예약/대여/출력) 선택 → 신청 완료!
FEATURES
주요 기능
🏠
실기실 예약
모형제작실, 3D프린팅실, 캐드실, 레이저커팅실 등 실기실 예약
🔧
물품 대여
전동공구, 수공구, 계측장비 등 실시간 재고 확인 후 대여
🖨️
출력 신청
Coated, Matt, Gloss, 흑백, 컬러 등 다양한 출력 요청
🖨️ 출력 신청 방법
PRINT GUIDE
1
사이드바에서 '출력 신청' 메뉴를 클릭합니다.
2
출력 용지 사이즈와 재질를 선택합니다. (Coated, Matt, Gloss)
3
출력 매수를 입력합니다.
4
출력할 파일과 입금내역을 업로드합니다. (PDF, 이미지 파일 등)
5
'신청하기' 버튼을 클릭하면 신청이 완료됩니다.
6
근로학생이 출력 처리 후 상태가 '완료'로 변경되면 수령하세요. (완료 이메일이 메일로 발송됩니다.)
🔧 물품 대여 방법
RENTAL GUIDE
1
사이드바에서 '물품 대여' 메뉴를 클릭합니다.
2
대여 가능한 물품 목록에서 원하는 물품을 확인합니다. (재고 실시간 표시)
3
물품을 선택하고 대여 수량과 반납 예정일을 입력합니다.
4
'대여 신청' 버튼을 클릭하여 신청을 완료합니다.
5
근로학생 확인 후 물품을 수령합니다. (복지관 602호)
6
반납 기한 내 반납해 주세요. 기한 초과 시 연체 경고가 부여됩니다.
자세한 사항은 포털사이트에서 확인해 주세요!
